# Easy
Easy is RSSReader that made by Model-View-Presenter.

## Architecture
```
|-- model
    |-- entity
        |-- xml
    |-- net
        |-- rss
    |-- ollie
    |-- repository
|-- presenter
|-- view
    |-- activity
    |-- adapter
    |-- component
    |-- popup
```
